# EmoGuard
一款基于Android Studio 的移动APP，与摄像头连接实现工业生产环境下的情绪感知与疲劳检测。

## 主要功能
- 用户的登录和注册。
- 对工厂作业人员的实时监控，并进行情绪感知和疲劳检测。
- 管理检测的指标。

## 开发工具
- Android Studio Giraffe | 2022.3.1 Patch 3
- VS-Code

## 部署
- 在Android Studio 中创建一个 bottom navigation project，将仓库中的res导入，替换掉系统自动生成的文件。
